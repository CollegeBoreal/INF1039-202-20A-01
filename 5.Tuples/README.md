# 5. Tuples


:m: Programmer une structure de données complexe, les `tuples` ou couples


Dans la lecture de l'`Unité 2` du livre, montrez quelques examples de `tuples`.

Par example, la différence entre un charactère et un `tuple`:

```python
a = 'A'
print(a)

etudiant = ( "Moi", 30000000, "Apprenant")
print(etudiant[2])
```
